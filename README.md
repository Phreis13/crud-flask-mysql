# CRUD de Usuários - Flask + MySQL

Projeto web desenvolvido em Python utilizando Flask e MySQL.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Exclusão de usuários

## Tecnologias
- Python
- Flask
- MySQL
- Bootstrap

## Como executar

python -m venv venv  
venv\Scripts\activate  
pip install flask mysql-connector-python  
python app.py  

Acesse: http://127.0.0.1:5000
